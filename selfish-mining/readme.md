# 针对矿池的一款selfish-mining程序
## 声明
```
Don't be evil, 技术分享
```
## 原理
1. 同时启动矿池挖矿和官方挖矿（或其他挖矿程序）
2. 当未找到答案时，提交到矿池
3. 当找到答案时，break矿池的提交，提交到官方挖矿，
   待官方挖矿成功后，continue矿池提交

## 矿池如何防范
1. 矿池在进行签名授权的时候，记录用户的pool_pk
2. 同时扫描block, 当block中出现的pool_pk是用户登记的pool_pk, 同时此块
的奖励地址不是矿池的，对此用户进行惩罚
